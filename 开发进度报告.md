# JWordCards 开发进度报告

## 项目概述

**项目名称**: JWordCards - 轻量级日语单词学习工具  
**技术栈**: Electron + React + TypeScript + Vite + TailwindCSS + Framer Motion  
**目标平台**: macOS / Windows 桌面应用
